# Prompt Optimizer Command

사용자의 프롬프트를 분석하고 Human-in-the-Loop 방식으로 최적화합니다.

## 입력 프롬프트
$ARGUMENTS

## 동작 방식

### 1단계: 프롬프트 평가
입력된 프롬프트의 품질을 0-100점으로 평가합니다.

**평가 기준:**
- 명확성: 요청이 구체적이고 명확한가?
- 맥락: 충분한 배경 정보가 제공되었는가?
- 목표: 원하는 결과물이 분명한가?
- 제약조건: 필요한 제한사항이 명시되었는가?

### 2단계: 점수별 모드 분기

**60점 미만 (직접 입력 모드):**
- 프롬프트 보충에 필요한 내용을 구체적으로 안내
- 사용자에게 직접 텍스트 입력을 요청
- 입력된 내용을 반영하여 프롬프트 개선

**60점 이상 (객관식 모드):**
- 프롬프트 개선을 위한 객관식 질문 생성
- 4가지 선택지 제시
- 사용자 선택을 반영하여 프롬프트 개선
- 최대 5회 반복

### 3단계: 종료 조건
- 점수가 90점 이상에 도달
- 또는 객관식 질문 5회 완료

### 4단계: 최종 확인
- 최적화된 최종 프롬프트 출력
- Y/N 확인 후 해당 프롬프트로 LLM 채팅 실행

## 실행 방법
```bash
cd ${CLAUDE_PLUGIN_ROOT}/scripts
uv run python main.py
```

## 요구사항
- OPENAI_API_KEY 환경변수 설정 필요
- Python 3.11+
- uv 패키지 매니저
